# 🧮 Smart Calculator

A modern and responsive calculator built using HTML, CSS, and JavaScript.

This project was developed as part of my web development learning journey to strengthen my understanding of JavaScript DOM manipulation, event handling, and responsive UI design.

---

## 📌 Project Overview

Smart Calculator is a simple yet elegant web application that performs basic arithmetic operations through an interactive user interface.

The project focuses on creating a clean user experience while implementing calculator logic using JavaScript.

---

## ✨ Features

* Addition, Subtraction, Multiplication, and Division
* Percentage (%) calculations
* Clear All (AC) functionality
* Delete Last Character (DEL) functionality
* Responsive design for mobile and desktop devices
* Modern glassmorphism user interface
* Smooth hover and click animations
* Real-time calculation results

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ Functionalities

### Number Input

Users can enter numbers using calculator buttons.

### Arithmetic Operations

Supported operators:

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)

### AC Button

Clears the entire display.

### DEL Button

Removes the last entered character.

### Equal (=)

Evaluates the entered mathematical expression and displays the result.
